# Separators

Porous separator membranes placed between electrodes. Separator pore volume contributes to the electrolyte fill calculation.

```python
separator = ocd.Separator(
    material=separator_material,
    thickness=25,    # μm
    width=310,       # mm
    length=5000,     # mm
)
```
