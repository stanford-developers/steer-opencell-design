# Electrolytes

Liquid electrolyte materials used to fill electrode assemblies. The electrolyte fills the pore volume of the assembly plus an overfill percentage.

```python
electrolyte = ocd.Electrolyte(
    name="1M LiPF6 in EC:DMC (1:1)",
    density=1.2, specific_cost=15.0, color="#00FF00",
)
```
